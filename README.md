# kcsoc-server

This is a repository containing a REST API part of the kcsoc-website backend.

It is currently being hosted at https://server.kcsoc.com/

## To Run

### With Docker

```bash
cp .env.sample .env
# Fill in .env file

docker-compose up -d
```

### Manually

```bash
cp .env.sample .env
# Fill in .env file

npm install
npm start
```
